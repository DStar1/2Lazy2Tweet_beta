# 2Lazy2Tweet_beta

### Setup:
Add a keys.js to the config file with your credentials
```
module.exports = {
    MongoURI: <MONGODB URI >,
    consumerToken: < YOUR CONSUMER TOKEN >,
    consumerTokenSecret: < YOUR CONSUMER TOKEN SECRET >
}
```

### Run:
Then to run app:
```
$> npm i
$> npm run start
```
